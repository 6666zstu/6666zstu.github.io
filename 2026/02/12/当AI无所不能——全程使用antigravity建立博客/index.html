当AI无所不能——全程使用antigravity建立博客网站

发表于 2026-02-12 更新于 2026-02-14 分类于 随笔 本文字数： 382 阅读时长 ≈ 1 分钟

📝 我的第一篇个人博客随笔

🎯 建站动机

我注意到不少的同龄人或者是一些大佬都搞了个博客出来，在里面分享自己的内容。在初步体验 google antigravity 的强大能力后，我决定也发布一个自己的博客。这是我在 AI coding 模式下的第一个 project。

⚡ 效率秒杀

使用 antigravity 调用 anthropic 最新的 claude opus 4.6 thinking 确实是一种秒杀的感觉。我在一开始设置时还出现了问题，导致所有的 run 还得手动确认。但是半个小时 claude 就基本给出了大的框架，一个小时内就已经大体弄完了（中间我自己还耽误了挺久），这个效率简直过于惊为天人。

🧩 遇到的问题

即便如此，claude 在应对一些小微场景还是展现出了问题。比如为了我博客背景图片的信息，我与 opus 4.6 互耗一个小时。我反思后得出结论：plan mode 和强大的模型不适合非 0-1 build 一个新项目。浪费 tokens 浪费时间。所以我换成 fast 的 sonnet 4.5，问题马上解决了，也是有点无语。

🔮 未来规划

这才是我写的第一篇真正意义上的博客（前面那个是 AI 写的）。我需要熟悉 markdown 格式。文章的话，可能会和公众号同步。后续再说，敬请期待，过年各家厂商开始搞事情了我这不冲浪一下。
