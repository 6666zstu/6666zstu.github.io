“高效编程”，其实是新型成瘾

发表于 2026-02-16 分类于 AI 本文字数： 4.7k 阅读时长 ≈ 17 分钟

6 个月，7 天，每天 12-14 小时。我没有熬夜，也没有荒废工作。我只是不再运动，不再娱乐，不再和朋友见面，不再探访家人。我把所有业余时间都交给了 AI 和那些永远不会上线的原型。我以为自己在“高效学习”，实际上只是在用生产力的幻觉，填补不敢面对真实生活的恐惧。那一刻我意识到：Vibe Coding成瘾和赌博、色情成瘾别无两样——它们都让你相信“我在做正确的事”，同时悄无声息地偷走你的人生。这篇文章是关于我们如何在“进步”的名义下，一点点失去自己的。

（非原创）

你以为自己在追求快乐，实际上你在逃避痛苦。
这是所有成瘾者共同的谎言——一个如此精巧的自我欺骗，以至于可能花费数年时间才能看清它的真面目。色情成瘾者点开“最后一个视频”，赌徒按下“最后一次”旋转按钮，Vibe Coder 敲下“最后一个提示词”，他们都相信这一次会不同。但大脑深处，同一套古老的神经回路正在运转，驱使他们重复同样的循环。
2025 年初，当 Andrej Karpathy 提出“Vibe Coding”这个概念时，没人预料到它会成为程序员群体中的新型成瘾。用自然语言描述需求，AI 瞬间生成代码，反复迭代直到“感觉对了”——这听起来像是生产力革命，却意外复制了老虎机和色情网站的成瘾机制。
这三种看似毫无关联的行为，实际上共享同一个底层架构：它们都劫持了人类大脑对不确定性奖励的原始渴望。

一、所有成瘾都服务于一个无意识目标

《How to fix your entire life in 1 day》中有一个观点：所有行为都是目标导向的，包括那些看起来在伤害你的行为。
当你凌晨三点还在刷色情网站，你并非因为“缺乏自制力”，而是在追求一个无意识目标——可能是逃避明天的工作压力，可能是填补深夜的孤独感，也可能是保护自己免受真实亲密关系的脆弱。
当赌徒明知会输光还要下注，他追求的表面是金钱，但潜藏深层目标其实是那几秒钟“命运悬而未决”的刺激感——在那个瞬间，他感到自己还活着，还有掌控感。
当 Vibe Coder 通宵迭代一个永远不会上线的原型，他追求的也不是产品，而是“几秒钟内想法变成现实”的魔法感——那种即时满足远比传统编程的缓慢积累更能缓解他对职业焦虑的恐惧。
成瘾的本质并非行为本身，而是你用它来解决什么问题。
问题在于，这些“解决方案”都是虚假的。它们提供短暂的缓解，却让真实问题越来越严重。你越是用色情逃避孤独，就越难建立真实联结；你越是用赌博寻求掌控感，就越感到人生失控；你越是用 Vibe Coding 追求即时成就，就越丧失深度思考的能力。
这就是成瘾的陷阱：它承诺解放，却带来囚禁。

二、间歇性强化：大脑最难抗拒的毒药

如果你想设计一个让人类上瘾的系统，你需要的只有一个原理：变比强化（Variable-Ratio Reinforcement）。
这是行为心理学中最强大的条件反射机制。既不是每次都给奖励（那会让人腻），也不是从不给奖励（那会让人放弃），而是不可预测地给奖励——有时大胜，有时小赢，有时失败，但你永远不知道下一次会是什么。
老虎机是这个原理的教科书案例。你拉一次杆柄：
可能中大奖（多巴胺爆棚）
可能小赢一点（维持希望）
可能什么都没有（但你想“下一次就会中”）
大脑在这种不确定性中疯狂分泌多巴胺，因为你期待可能得到奖励。这种期待本身就是毒药。
色情：无限新鲜感的陷阱
色情网站完美复制了这个机制。每次点击“下一个视频”，你不知道会看到什么——可能是强烈的兴奋，也可能是平淡的失望。但那种“下一个可能更刺激”的幻觉，让你不断追逐。
更可怕的是，互联网色情提供了无限的新鲜感。在人类进化史上，性资源是稀缺的，大脑的奖励系统被设计为“遇到新的性机会时大量释放多巴胺”。但现在，无限的新面孔、新场景、新类型，让大脑误以为你正在经历人类历史上最成功的繁殖季节。
结果是：你永远得不到满足。因为满足来自于“达成目标”，但当目标是“无限的新鲜感”时，终点永远不会到来。
赌博：永远“就差一点”
赌博的间歇性强化更加精密。现代老虎机和赌博 App 经过精心设计，让你频繁地“差一点就赢”——三个图标中了两个，差一点就中大奖。
这种“near miss”（接近成功）比完全失败更能驱使你继续。因为大脑把它解读为“我快成功了，再试一次就行”。研究显示，赌徒在“差一点赢”时的大脑活动，和真正赢钱时几乎一样强烈。
加密货币短线交易、盲盒抽奖、游戏开箱——所有这些现代赌博变体都在复制同一套逻辑：让你永远处在“就差一点”的边缘。
Vibe Coding：程序员的老虎机
Vibe Coding 的成瘾性让整个开发者社区措手不及。
你向 AI 提示一次：
可能生成完美的功能，一次运行成功（大胜，兴奋感爆棚）
可能 90% 正确，有小 bug（小挫败，但你想“再提示一次就修好了”）
可能完全崩溃（惩罚，但你已经上头，继续迭代）
这和老虎机的逻辑完全一致。
更致命的是即时反馈。传统编程需要深度思考、缓慢调试，可能几小时才看到结果。但 Vibe Coding 几秒钟就能看到想法变成运行的 App——那种“魔法感”触发的多巴胺释放，远超传统编程的缓慢成就感。
许多开发者描述：“开始只是想快速做个原型，结果通宵迭代，忽略了工作、睡眠和伴侣。我知道这些代码质量很差，我根本没读懂，但就是停不下来。”
一位程序员在 Reddit 上写道：“Vibe Coding 就是程序员版的 gooning（一种长时间边缘化追求兴奋的性行为），你是在追逐下一次多巴胺冲击。”

三、HALT：成瘾的四个触发器

成瘾恢复领域有一个经典工具：HALT。它代表四个最危险的生理和情绪状态：
Hungry （饥饿）
Angry （愤怒）
Lonely （孤独）
Tired （疲惫）
当你处于这四种状态之一时，大脑的前额叶（负责理性决策和自控）功能会显著下降，而边缘系统（负责即时奖励和情绪反应）会接管控制权。这时，成瘾行为最容易被触发。
为什么 HALT 如此危险？
饥饿：血糖降低导致情绪调节中枢（杏仁核）过度活跃，理性决策能力下降。你更容易冲动行事。
愤怒：应激激素（皮质醇、肾上腺素）飙升，大脑进入“战斗或逃跑”模式。成瘾行为成为“灭火”的快速方式。
孤独：催产素和多巴胺系统失衡。大脑把色情、赌博、Vibe Coding 当作“虚假联结”的替代品——它们提供短暂的陪伴感，却让真实孤独越来越深。
疲惫：睡眠不足会削弱前额叶的抑制功能，自控力几乎归零。深夜是所有成瘾行为的高发时段。
最危险的组合
当两个或更多 HALT 状态同时出现时，复发风险呈指数级上升。
孤独 + 疲惫是色情成瘾最常见的触发组合。深夜一个人，疲惫但睡不着，打开手机“就看一眼”——这个场景在无数成瘾者的故事中重复出现。
愤怒 + 疲惫是赌博成瘾的高危组合。工作受挫，身心俱疲，想要“赢一把”来证明自己还有掌控力。
孤独 + 疲惫同样是 Vibe Coding 成瘾的温床。加班到深夜，没人聊天，打开 AI 工具“建个酷炫项目”，瞬间获得目的感和“与 AI 对话”的虚假连接。
成瘾非道德问题，而是大脑在 HALT 状态下用错误方式求救。

四、身份陷阱：你捍卫的不是行为，而是自我

一个更深层的机制：身份循环：
你设定一个目标（可能是无意识的）
通过该目标的透镜感知现实
只注意“重要”信息（学习）
行动并获得反馈
重复直至自动化（条件反射）
行为成为身份认同（“我是那种……的人”）
你捍卫这个身份以保持心理一致性
身份塑造新目标，循环重启
这个循环解释了为什么戒除成瘾如此困难：对抗一个行为很容易，但是对抗一个身份很困难。
色情成瘾者可能无意识地认同“我是那种需要性幻想来应对压力的人”。赌徒可能认同“我是那种敢于冒险的人”。Vibe Coder 可能认同“我是那种快速迭代、拥抱 AI 的现代开发者”。
当你试图改变行为时，大脑会感到身份受到威胁——就像身体受到威胁一样，它会启动“战斗或逃跑”反应。你会找一千个理由为自己辩护：
“我只是压力大，放松一下而已”（色情）
“我只是在研究概率，不是真的赌博”（赌博）
“我只是在学习新工具，这是生产力投资”(Vibe Coding)
这些层层叠加，形成了身份保护机制。
更可怕的是，这个身份往往在童年就开始形成。如果你从小缺乏即时奖励或掌控感，成年后遇到提供“即时满足”的成瘾行为时，大脑会特别容易上钩——因为它终于找到了童年缺失的东西。

五、子人格：成瘾者内心的另一个声音

成瘾恢复专家乔治·柯林斯在《Breaking the Cycle》中提出一个有力的比喻：成瘾就像一个独立的子人格。
它不是真正的“你”，而是“你内心的另一个声音”。这个声音有自己的逻辑、自己的目标、自己的话术：
色情成瘾的子人格说：
“就看一眼，不会怎么样的”
“你今天压力这么大，应该奖励自己”
“反正没人知道，不会伤害任何人”
赌博成瘾的子人格说：
“这次感觉不一样，我有预感会赢”
“刚才差一点就中了，说明我快成功了”
“输的钱必须赢回来，不然就白输了”
Vibe Coding 成瘾的子人格说：
“再提示一次，就完美了”
“这个功能超级酷，为什么要停？”
“我在学习新技术，这是投资未来”
这些声音听起来如此合理，以至于你很难分辨它们不是“真正的你”。但如果你仔细观察，会发现它们有一个共同特征：它们总是在你脆弱的时候出现，总是提供短期解决方案，总是忽略长期代价。
恢复的第一步不要急着“战胜”这个声音，而是学会识别它。当你能在它出现时说“哦，这是那个声音又来了”，你就有了选择的空间。

六、虚假生产力：最隐蔽的逃避

Vibe Coding 比色情和赌博更危险的一点在于：它披着生产力的外衣。
色情和赌博很难自我欺骗——你知道自己在做什么，只是无法停止。但 Vibe Coding 可以伪装成“工作”。
你可以对自己说：
“我在构建产品”
“我在学习 AI 工具”
“我在提升效率”
但如果你诚实地审视，会发现：
你构建的“产品”都是永远不会上线的原型
你“学习”的是如何更好地提示 AI，而不是理解代码本身
你的“效率”是产出大量低质量代码，而非解决真实问题
一位开发者的自白令人震撼：“我一天能用 Vibe Coding 建三个 App，但一个月后回头看，没有一个是有用的。我逃避的并非工作，而是真正的深度思考——那种需要面对自己无知、需要缓慢学习、需要接受挫败的过程。Vibe Coding 让我感觉自己很厉害，但实际上我的技能在退化。”
这种“虚假生产力”在现代社会无处不在：
刷职业发展类短视频，而不是真正学习
收藏一百篇文章，而不是读完一篇
参加无数线上课程，而不是完成一个项目
它们都提供“我在进步”的幻觉，却让你逃避真正的成长。

七、打破循环：从觉察到重建

如果你读到这里，发现自己陷入了某种成瘾，请记住：这是大脑被精心设计的奖励系统劫持了。
恢复不能靠“意志力”，而是靠重建整个系统。
第一步：觉察触发
每次冲动来临时，立刻暂停，问自己：
我现在 HALT 了吗？
饥饿？→ 先吃一顿均衡的饭
愤怒？→ 承认情绪，用健康方式释放（写下来、运动、深呼吸）
孤独？→ 主动联系一个真实的人
疲惫？→ 优先休息，不要和冲动谈判
大多数冲动的根因并非“我想做这件事”，背后潜藏着“我现在 HALT 了，大脑在用老办法求救”。
第二步：质疑第一个念头
子人格的声音总是第一个出现。学会识别它：
“就最后一次” → 这是谎言，从来没有“最后一次”
“我应该奖励自己” → 真正的奖励不会让你第二天后悔
“再试一次就好了” → 这是间歇性强化在说话
在冲动和行动之间，插入 30 秒的缓冲时间。大多数冲动会在 30 秒内减弱。
第三步：改变深层目标
记住：真正的改变需要改变目标和身份，而非仅仅改变行为。
问自己：
我用这个成瘾行为在逃避什么？
我真正需要的是什么？（真实联结？深度成就？情绪疗愈？）
如果我是“已经戒除的人”，我会如何度过今天？
第四步：追求真实满足
成瘾提供的是虚假满足——短暂的高潮，却让真实需求长期匮乏。
色情成瘾者真正需要的并非性刺激，而是真实的亲密联结、被看见、被接纳。
赌徒真正需要的并不是金钱，而是掌控感、成就感、生活的意义。
Vibe Coder 真正需要的也不是快速原型，而是扎实的技能、深度的理解、可持续的创造力。
恢复的终极目标非“戒除某个行为”，而是建立一种不需要不断“再来一次”就能感到完整的生活。
第五步：重建身份
构建反愿景：如果继续成瘾，5 年后、10 年后、一生后你会成为什么样的人？让这个画面足够清晰、足够痛苦，以至于你无法容忍。
构建最小可行愿景：你真正渴望的生活是什么样的？不去想“戒除 XX”，而是“成为能够 XX 的人”。
设定身份目标：
不是“我要戒色情”，而是“我是那种能够建立真实亲密关系的人”
不是“我要戒赌博”，而是“我是那种通过扎实努力创造价值的人”
不是“我要少用 AI”，而是“我是那种深度理解代码、追求工程卓越的开发者”
每日行动：那个“已经恢复的你”今天会做什么？时间块安排 2-3 个具体行动。
第六步：将生活游戏化
把恢复变成一个游戏：
反愿景 = 赌注（如果失败会怎样）
愿景 = 胜利条件
30 天清醒 = 主线任务
每次成功抵御冲动 = 小任务（获得经验值）
HALT 检查、联系支持者 = 日常任务
约束（不在深夜独处、不带手机进卧室）= 游戏规则
这不能把恢复变得“有趣”，但是能给大脑提供新的奖励系统——用真实的进步替代虚假的高潮。

八、成瘾的终极谜底

色情、赌博、Vibe Coding——三种看似不同的成瘾，实则共享同一个谜底：
你并非在追求快乐，而是逃避成为真正的自己。
成瘾提供了一个完美的借口：只要你还在循环中，你就不必面对那些真正困难的问题——
如何建立真实的亲密关系（需要脆弱和被拒绝的风险）
如何创造真正的价值（需要缓慢积累和延迟满足）
如何成为真正的专家（需要面对无知和持续学习）
成瘾让你停留在“永远在追逐下一次”的状态，而不必面对“我已经到达，现在该做什么”的空虚。
但这种逃避的代价是：你永远无法成为你本可以成为的人。
恢复的目的并非回到“正常”，而是要第一次成为真正的自己——那个不需要用虚假满足来填补空洞的人，那个能够面对真实生活的复杂和不确定的人，那个值得拥有真正快乐的人。
如果你正在挣扎，请记住：成瘾者不是“坏人”或“弱者”，他们只是掉进了大脑古老奖励系统与现代技术环境结合的陷阱。
你值得过一种不需要不断“再来一次”才能感到完整的生活。
恢复从承认“它在干扰我的生活”开始。
而改变从今天、从此刻、从下一个你选择不点击的瞬间开始。
